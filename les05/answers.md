## 1. Для чего используется диаграмма рассеяния?

**Диаграмма рассеяния (scatter plot)** используется для:
- Визуализации взаимосвязи между двумя непрерывными переменными
- Обнаружения корреляций между переменными
- Выявления кластеров и выбросов в данных
- Анализа распределения данных в двумерном пространстве

**Пример использования:** изучение зависимости роста от веса, анализ связи цены и качества товара.

## 2. Что такое гистограмма, чем она отличается от столбчатой диаграммы?

**Гистограмма** - показывает распределение непрерывной переменной, где:
- Ось X: интервалы значений (бины)
- Ось Y: частота попадания в интервал
- Столбцы соприкасаются (непрерывные данные)

**Столбчатая диаграмма** - показывает сравнение категориальных данных, где:
- Ось X: категории
- Ось Y: значения
- Столбцы разделены (дискретные данные)

**Основные различия:**
| Характеристика | Гистограмма | Столбчатая диаграмма |
|----------------|-------------|---------------------|
| Тип данных | Непрерывные | Категориальные |
| Столбцы | Соприкасаются | Разделены |
| Порядок осей | Упорядоченный | Любой порядок |

## 3. Как задать цвет линии в линейном графике Matplotlib?

```python
import matplotlib.pyplot as plt

# Способ 1: параметр color
plt.plot(x, y, color='red')

# Способ 2: сокращенный параметр
plt.plot(x, y, 'r-')  # красная линия

# Способ 3: RGB/HEX значения
plt.plot(x, y, color='#FF5733')
plt.plot(x, y, color=(0.1, 0.2, 0.5))

# Способ 4: именованные цвета
plt.plot(x, y, color='blue')
```

## 4. Чем класс plt.Figure отличается от объекта plt.Axes?

**plt.Figure** - это весь холст (окно), который может содержать несколько графиков.

**plt.Axes** - это отдельная область для построения одного графика на Figure.

**Аналогия:** Figure - это вся страница, Axes - отдельный рисунок на странице.

```python
# Создание Figure и Axes
fig = plt.figure()           # Создает холст
ax = fig.add_subplot(111)    # Создает область для графика

# Или сокращенно
fig, ax = plt.subplots()     # Создает и холст и область

# Figure управляет всем изображением
fig.set_size_inches(10, 8)
fig.suptitle('Общий заголовок')

# Axes управляет конкретным графиком
ax.plot(x, y)
ax.set_title('Заголовок графика')
ax.set_xlabel('Ось X')
```

## 5. Matplotlib не позволяет управлять типом маркера scatter графика. Да или нет?

**НЕТ.** Matplotlib позволяет управлять типом маркера в scatter графике.

```python
plt.scatter(x, y, marker='o')    # круги
plt.scatter(x, y, marker='s')    # квадраты
plt.scatter(x, y, marker='^')    # треугольники
plt.scatter(x, y, marker='*')    # звезды
plt.scatter(x, y, marker='+')    # плюсы
plt.scatter(x, y, marker='x')    # крестики
```

## 10. Как на одном холсте может располагаться не более 16 диаграмм, это обусловлено тем что размер диаграммы нельзя изменить и более мелкие графики попросту не будут видны.

**НЕТ.** Это утверждение неверно по нескольким причинам:

1. **Можно создавать больше 16 диаграмм:**
```python
# Создание 20 диаграмм (4x5)
fig, axes = plt.subplots(4, 5, figsize=(15, 12))
```

2. **Размер диаграммы можно изменять:**
```python
plt.figure(figsize=(20, 16))  # Большой холст для многих графиков
```

3. **Можно настраивать отступы и размеры:**
```python
fig, axes = plt.subplots(5, 5, figsize=(20, 16))
plt.subplots_adjust(wspace=0.4, hspace=0.4)  # Регулировка расстояний
```

**Ограничения** определяются скорее читабельностью, чем техническими возможностями Matplotlib.
