## 1. Что такое медиана? Рассчитать медиану для предложенного ряда чисел.

**Медиана** - это значение, которое делит упорядоченный набор данных пополам. 50% значений лежат ниже медианы, 50% - выше.

**Для нечетного количества элементов:** средний элемент упорядоченного ряда  
**Для четного количества элементов:** среднее арифметическое двух центральных элементов

**Пример расчета:** для ряда [3, 1, 4, 2, 5]
1. Упорядочиваем: [1, 2, 3, 4, 5]
2. Находим средний элемент: 3

**Медиана = 3**

## 2. Что такое дисперсия? Рассчитать дисперсию для предложенного ряда чисел.

**Дисперсия** - это среднее арифметическое квадратов отклонений значений от их среднего арифметического. Показывает меру разброса данных.

**Формула:** σ² = Σ(xᵢ - μ)² / N

**Пример расчета:** для ряда [3, 1, 4, 2, 5]
1. Среднее арифметическое: (3+1+4+2+5)/5 = 15/5 = 3
2. Квадраты отклонений:
   - (3-3)² = 0
   - (1-3)² = 4
   - (4-3)² = 1
   - (2-3)² = 1
   - (5-3)² = 4
3. Сумма квадратов отклонений: 0+4+1+1+4 = 10
4. Дисперсия: 10/5 = 2

**Дисперсия = 2**

## 3. Что такое среднеквадратичное отклонение? Рассчитать среднеквадратичное отклонение для предложенного ряда чисел.

**Среднеквадратичное отклонение (СКО)** - это квадратный корень из дисперсии. Показывает средний разброс значений вокруг среднего в тех же единицах измерения, что и исходные данные.

**Формула:** σ = √σ²

**Пример расчета:** для ряда [3, 1, 4, 2, 5]
1. Дисперсия (из предыдущего расчета) = 2
2. СКО = √2 ≈ 1.414

**СКО ≈ 1.414**

## 4. Что такое Series и чем он отличается от DataFrame

**Series** - одномерный помеченный массив, который может содержать данные любого типа.

**DataFrame** - двумерная помеченная структура данных с столбцами потенциально разных типов.

**Различия:**
- **Размерность:** Series - 1D, DataFrame - 2D
- **Структура:** Series - один столбец с индексом, DataFrame - таблица с строками и столбцами
- **Индексация:** Series имеет один индекс, DataFrame имеет индекс строк и имена столбцов

```python
import pandas as pd

# Series
s = pd.Series([1, 2, 3], index=['a', 'b', 'c'])

# DataFrame
df = pd.DataFrame({'col1': [1, 2, 3], 'col2': ['a', 'b', 'c']})
```

## 5. Можно ли изменить индекс в Series после создания объекта?

**Да, можно** несколькими способами:

```python
s = pd.Series([1, 2, 3], index=['a', 'b', 'c'])

# Способ 1: прямое присваивание
s.index = ['x', 'y', 'z']

# Способ 2: метод rename()
s = s.rename({'x': 'new_x', 'y': 'new_y'})

# Способ 3: метод set_axis()
s = s.set_axis(['p', 'q', 'r'])
```

## 6. Операции сложения над Series

```python
# Исходные данные
s1 = pd.Series([1, 2, 3], index=['a', 'b', 'c'])
s2 = pd.Series([4, 5, 6], index=['a', 'b', 'c'])
s3 = pd.Series([4, 5, 6], index=['x', 'y', 'z'])
s4 = pd.Series([1, 2], index=['a', 'b'])
s5 = pd.Series([1.5, 2.5, 3.5], index=['a', 'b', 'c'])
s6 = pd.Series(['x', 'y', 'z'], index=['a', 'b', 'c'])
```

**Ответы:**

1. **Одинаковые типы, размеры и индексы:** ✅ Да  
   `s1 + s2` → `[5, 7, 9]`

2. **Одинаковые типы, размеры, но разные индексы:** ✅ Да (но с NaN)  
   `s1 + s3` → `[NaN, NaN, NaN]` (индексы не совпадают)

3. **Одинаковые типы, разные размеры и одинаковые индексы:** ✅ Да (с выравниванием)  
   `s1 + s4` → `[2, 4, NaN]` (отсутствующие значения = NaN)

4. **Разные типы, одинаковые размеры и индексы:** ✅ Да (автоматическое приведение)  
   `s1 + s5` → `[2.5, 4.5, 6.5]` (int → float)

5. **Разные типы, одинаковые размеры и разные индексы:** ❌ Нет (ошибка или NaN)  
   `s1 + s6` → Ошибка или `[NaN, NaN, NaN]`

6. **Разные типы, разные размеры и разные индексы:** ❌ Нет  
   Результат непредсказуем или ошибка

## 7. Что такое DataFrame и чем он отличается от Series

**DataFrame** - это двумерная помеченная структура данных с столбцами потенциально разных типов, похожая на таблицу в SQL или электронную таблицу.

**Отличия от Series:**
- **Размерность:** DataFrame - 2D, Series - 1D
- **Индексы:** DataFrame имеет индекс строк и имена столбцов, Series - только индекс
- **Данные:** DataFrame может содержать разные типы в разных столбцах
- **Операции:** над DataFrame можно выполнять операции по строкам и столбцам

## 9. Чем отличается метод iloc от метода loc

| Характеристика | iloc | loc |
|----------------|------|-----|
| **Тип индексации** | По позициям (integer location) | По меткам (labels) |
| **Диапазон** | [start:end] - end исключается | [start:end] - end включается |
| **Пример** | `df.iloc[0:3]` - строки 0,1,2 | `df.loc['a':'c']` - строки a,b,c |

```python
df = pd.DataFrame({'A': [1, 2, 3, 4]}, index=['a', 'b', 'c', 'd'])

df.iloc[0:2]    # Строки 0 и 1 (позиции)
df.loc['a':'c'] # Строки a, b, c (метки)
```

## 10. Можно ли добавить в DataFrame дополнительный Series?

**Да, можно** несколькими способами:

```python
df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})
new_series = pd.Series([7, 8, 9], name='C')

# Способ 1: присваивание по имени столбца
df['C'] = new_series

# Способ 2: метод assign()
df = df.assign(C=new_series)

# Способ 3: конкатенация
df = pd.concat([df, new_series], axis=1)
```
